Found a bug?

You can report bugs below or on our github page

Bug Report Guidelines:
When reporting a bug, you should follow these guidelines:

  • Be specific as possible: How do I reproduce the bug, what causes the bug? Some helpful data would be links to screenshots, links to webpages where the extension fails, OS and browser version and version of the extension.
  • The extension makes mistakes: Because language is incredibly complex, to provide a quick summary the extension uses some shortcuts. Occasionally, this might cause erroneous data. Unless the error occurred multiple times in separate sources, you should not submit a bug report.
  • Don't Submit Suggestions: This is a bug report page. If you would like to suggest an enhancement or feature, please do so at our github page
This page offers more info on how to write good bug reports.
