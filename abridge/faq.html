Frequently Asked Questions

  • What Does Debug Mode Do? Debug mode adds more information on the inner workings of the extension, which you might find interesting (Or not)
  • What Does Search All Tags Do? This feature makes the extension search all the text on the webpage, rather than just paragraph tags. This is used on webpages that don't necessarily use paragraph tags. What's a paragraph tag?
  • What Does No Timeout Do?If you're on a very slow computer and it takes too long to summarize the webpage, the extension will ask if you want to exit. Enabling this will disable that popup.
  • What Does Search Hidden Do? Some webpages have invisible text that you aren't meant to see. You can enable this to have the extension analyze invisible text too, but it's not recommended.
